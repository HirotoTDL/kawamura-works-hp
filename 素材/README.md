## ディレクトリ構成

```
素材/
├── 画像/
│   ├── ロゴ/         ヘッダー/フッター用ロゴ
│   ├── トップ/       トップページ用メイン・キャッチ画像
│   ├── 会社案内/     外観・韓国協力工場
│   ├── 製品情報/     製品カテゴリ別サムネイル

│   └── ニュース/     ニュース記事用画像
├── 資料/             会社案内パンフレット(PDF/JPG)
├── テキスト/         各ページの本文(Markdown)
└── README.md         本ファイル
```
